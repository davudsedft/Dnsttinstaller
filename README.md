یک فایل داریم با نام
 dns.zip

اینو ببرین توی سرور
میتونین اپلودش کنین یجا
و با دستور
wget https://youradress/dns.zip

دانلودش کنین
یا با دستورات ssh
برای انتقال فایل راه زیاده
وقتی فایل رو بردین روی سرور لینوکسی
ابتدا جایی که فایل dns.zip
هست باید unzip بشه
کاری نداره
ابتدا نصب unzip
apt install unzip

بعد از نصب unzip
اینجوری unzip میشه

unzip dns.zip
و با اینتر دوتا فایل خارج میشه

یکی اسمش هست
dnstt-server
و یکی هست
dnstt-setup.sh

دوتا دستور ساده برای دسترسی

chmod +x dnstt-server
chmod +x dnstt-setup.sh

و در نهایت
bash dnstt-setup.sh
و اینتر
منو شماره یک و دو داره

میگه میخاین نصب یا حذف کنین
یک نصب و دو حذف dnstt هست
یک وارد و اینتر
حالا از شما 
دامنه میخاد
دامنه رو وارد و اینتر

اینتر سوم پورت برای فورادر داخلی دی ان اس از پورت ۵۳
که میتونین ۵۳۰۰ بزنین

اینتر بعدی ساخت  پورت ساکس برای ارتباط سرور بدون ssh که میتونین ۸۰۰۰
و با اینتر اخر
لینک dns:// ساخته میشه
لینک و ببرین تو purvpn
